# Tyria's GPS

Tyria's GPS is the starting structure for this module.

## Current Stage

- Core project files are in place.
- Default corner icon is available.
- Search window with a search box is included but not usable in the present state.
- Logging support is included for development testing.

## Installation

1. Download the module .bhm file.
2. Open BlishHUD.
3. Open the Modules window.
4. Right-click inside the Modules window and choose the option to open your module folder.
5. Copy the .bhm file into that folder.
6. Back in BlishHUD, refresh modules (or restart BlishHUD).
